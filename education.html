Full-stack Webdeveloper Opensource

Do you want to work on developing modern interactive websites, web applications or e-commerce systems? As a full-stack web developer, you work to develop advanced, dynamic and user-friendly web solutions. This is the education for you who like creative problem solving and who want a broad web developer competence in open source and with knowledge in both backend and frontend! As a web developer, you write the code run by browsers for new web pages and maintain and update existing web pages for both large and small web systems. The training gives you a broad knowledge in programming and specialized knowledge in the browser's conditions, web standards and specifications. The training is aimed at those who like to collaborate and together find creative solutions and drive projects forward.
